Now that we have our transformed principal components, it's a nice idea to check out the weight of each variable on the first few components to see if they can be interpreted in some fashion.

The further a weight is from zero, the more the principal component is in the direction of the corresponding feature. If two features have large weights of the same sign (both positive or both negative), then increases in one tend expect to be associated with increases in the other. def clusterit(df,SRange,ERange):
    avgs=[]
    start_time = time.time()
    rng = range(SRange,ERange,2)
    for i in rng:
        # run k-means clustering on the data and...
        kmeans = KMeans(n_clusters=i).fit(df)

        # Obtain a score related to the model fit
        score = np.abs(kmeans.score(df))

        # compute the average within-cluster distances.
        avg =score/df.shape[0]
        print("For number of clusters equals to {} , average within-cluster distances is {}s".format(i,avg))
        avgs.append(avg)

    print("Run Time -- {}m ".format(np.round((time.time() - start_time)/60,2)))

    return rng ,avgs
